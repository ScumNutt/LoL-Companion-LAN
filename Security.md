# Security Notes

## Network Model

League Companion is designed for trusted local networks.

The connection exists between:

PC ↔ User Device

Do not expose the application directly to the public internet.

---

## Permissions

The application requires:

- Local network communication
- Access to its own files
- Running while League is open


It does not require:

- Administrator privileges
- Driver installation
- Anti-cheat access

---

## Privacy

League Companion:

- Does not use cloud accounts
- Does not collect telemetry
- Does not store credentials

---

## Threat Model

The application assumes:

- You trust devices on your local network
- You control the connected phone/device

Use only on networks you trust.
