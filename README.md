# LoL-Companion-LAN

A local network companion application that allows you to interact with League of Legends client features from another device.

No cloud. No account. No game modification.

## What is it?

League Companion runs on your PC and creates a private connection with your phone over your local network.

Your phone acts as a second interface for the League client.

Designed for convenience:
- Keep your game window focused
- View lobby information
- Manage client actions from another device
- Use a cleaner second-screen experience

The League client must be running on your PC.

---

## Features

✅ Phone ↔ PC local network connection  
✅ Works through your browser (no mobile installation required)  
✅ QR pairing for quick setup  
✅ Lobby information display  
✅ Queue controls  
✅ Role preferences  
✅ Champion select interaction  
✅ Pick / ban / hover actions  
✅ No account required  
✅ No cloud services  

---

## What this project does NOT do

League Companion is NOT a bot and does NOT provide gameplay advantages.

❌ No automation  
❌ No unattended actions  
❌ No scripts  
❌ No macros  
❌ No memory reading or writing  
❌ No DLL injection  
❌ No game file modification  
❌ No anti-cheat interaction  
❌ No screen capture  
❌ No keyboard/mouse simulation  
❌ No Riot account credentials stored  

Every action requires explicit user interaction.

## How it works
Phone
|
| Local Wi-Fi network
|
League Companion PC App
|
| Local client connection
|
League Client


The application:

- Runs locally on your computer
- Communicates with your phone over your private network
- Uses the running League client interface
- Does not route traffic through external servers

---

## Installation

1. Download the latest release
2. Extract the application
3. Run League Companion on your PC
4. Open the displayed QR code/link on your phone
5. Make sure both devices are connected to the same network

---

## Security Notes

League Companion is designed for trusted local networks.

The application:

- Does not create an online account
- Does not upload your game data
- Does not store Riot credentials
- Only communicates inside your local environment
